\# Atelier 4 - Gestion de voitures



\## Description

Ce projet Python simule la gestion d’un parc de voitures de service en utilisant la programmation orientée objet.



Le programme contient deux classes :

\- Employe

\- Voiture



Il permet de créer des employés, créer des voitures, attribuer une voiture à un employé et afficher leurs informations.
